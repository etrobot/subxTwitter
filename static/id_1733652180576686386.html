06 Jan 2024 08:09:22@_philschmid: Kami mendapat hadiah Natal terlambat dari @Microsoft! 🎁🤗 Microsoft baru saja mengubah lisensi untuk model LLM kecil mereka phi-2 menjadi MIT! 🚀

Phi-2 adalah model dasar 2,7 miliar parameter yang dilatih pada 1,4T token, termasuk data sintetis, mencapai 56,7 pada MMLU, mengungguli Google Gemini Nano.

TL;DR:
🧮 2,7B Parameters base model
📱 ~1GB memory requirement for quantization
📜 context length of 2048 tokens.
🚀 outperforms Google Gemini Nano (3.2B)
🧑🏻‍💻 Bagus dalam coding, terutama Python
🗣️ Dilisensikan di bawah MIT
🤗 Tersedia di Hugging Face
